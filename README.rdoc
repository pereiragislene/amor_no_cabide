== README

O Amor no Cabide é um projeto que estimula a doação de roupas em Porto Alegre e está crescendo para outros estados.

Criei este site para ajudar pessoas a encontrarem os pontos onde existem cabides, e permitir que novos pontos sejam cadastrados.

* Tech Stack:
- Ruby 1.9.3
- Rails 4.1.1
- ActiveAdmin (Gem para administração de pontos e usuários)
- Bootstrap-generators (Gem)
- Google Maps API

Site disponível em: http://amornocabide.herokuapp.com/
